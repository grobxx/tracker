# Отчет по проекту: Перевод 100% продуктового трафика на новую архитектуру защиты веб-приложений

## Общая оценка состояния проекта

### Статус и отставание от графика
**Статус:** Проект находится в стадии `at_risk`. Ответственный за управление проектом Aleksey Gimatdinov отметил, что проект не соответствует срокам выполнения.

### Ключевые риски
1. **Риск критического ухудшения уровня безопасности при сокращении поставщиков**
   - **Вероятность:** Высока
   - **Влияние:** Критическое для бизнеса, если не будет обеспечена достаточная защита.
   
2. **Сложность миграции 100% трафика**
   - **Вероятность:** Высокая
   - **Влияние:** Могут возникнуть проблемы с доступностью и функциональностью сервисов во время миграции.

3. **Риск ухудшения инцидент-менеджмента в переходный период**
   - **Вероятность:** Высокая
   - **Влияние:** Возможное увеличение времени реакции на инциденты и потери доверия клиентов.

4. **Риск роста стоимости при переходе**
   - **Вероятность:** Средняя
   - **Влияние:** Если стоимость вырастет, это может негативно сказаться на финансовой перспективе компании.
